PSICK

Puppet Systems Infrastructure Construction Kit

PSICK is a full-featured, reusable, modern Puppet control-repo

It features:
  • Integrations with Vagrant, Docker, GitLab, Fabric and other tools.
  • A complete, entirely data-driven, set of Puppet profiles and tools for common use cases.
  • Toolset and integrations for full Puppet life cycle management: issue tracking, development, testing, CI, deployment.
We offer the following PSICK services:
  • PSICK KickOff, where a POC Puppet Infrastructure is designed in a training on the job way.
  • Remote Help Line for PSICK setup and customisation.
  • Support plans for control-repo code review and update.
